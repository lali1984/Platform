# Руководство по внесению вклада

## Процесс разработки

1. Создайте форк репозитория
2. Создайте ветку для вашей функции
3. Внесите изменения
4. Напишите тесты (если необходимо)
5. Убедитесь, что все тесты проходят
6. Создайте Pull Request

## Code Style

Мы используем ESLint и Prettier для поддержания единого стиля кода.

```bash
npm run lint
