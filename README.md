# Platform Ecosystem

Микросервисная образовательно-предпринимательская платформа.

## Описание

Platform Ecosystem — это комплексная микросервисная платформа, объединяющая образовательные ресурсы, инструменты для стартапов и коммуникационные сервисы в единую экосистему.

## Архитектура

Платформа построена на микросервисной архитектуре с использованием:
- Node.js (TypeScript) для сервисов
- PostgreSQL для хранения данных
- Redis для кэширования и сессий
- Kafka для асинхронной коммуникации
- Docker и Docker Compose для контейнеризации
- Nginx в качестве API Gateway
